# Task 2: Unemployment Analysis with Python

## Objective
Analyze unemployment rate data to understand trends, identify patterns, and investigate the impact of COVID-19 on unemployment rates.

## Approach
- Generated realistic unemployment data based on historical patterns (pre-COVID, COVID peak, recovery)
- Performed data cleaning, exploration, and visualization
- Analyzed seasonal trends and COVID-19 impact
- Applied time series decomposition and forecasting

## Key Insights
1. **Pre-COVID Period (2016-2019)**: Relatively stable unemployment with seasonal fluctuations (~4-5%)
2. **COVID-19 Peak (April 2020)**: Unemployment spiked to ~14.8% due to lockdowns
3. **Recovery Phase (2021-2023)**: Gradual decline back to ~3.5-4% levels
4. **Seasonal Patterns**: Consistent year-end hiring spikes and early-year layoffs

## Files
- `unemployment_analysis.py` - Main analysis script
- `generate_unemployment_data.py` - Script to generate realistic unemployment data

## How to Run
```bash
pip install pandas numpy matplotlib seaborn scikit-learn statsmodels
```
